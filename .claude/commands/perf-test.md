# Performance Test

运行完整的性能测试套件，验证 SenseFlow 的性能指标。

## 测试项目

### 1. CPU 占用测试
- **目标**: < 0.1%
- **方法**: 监控应用 5 分钟后台运行的 CPU 占用

### 2. 数据库查询性能
- **目标**: < 50ms
- **测试**: 搜索查询、历史记录读取、插入操作

### 3. 搜索响应速度
- **目标**: < 10ms
- **测试**: 实时过滤 200+ 条记录

### 4. 动画帧率
- **目标**: 60fps
- **测试**: 窗口显示/隐藏、卡片滚动

### 5. 内存使用
- **测试**: 200+ 条记录后的内存占用

## 使用方法

```
/perf-test
```
