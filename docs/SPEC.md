# SenseFlow - 产品需求规范

**版本**: v0.4
**平台**: macOS 14.0+
**技术栈**: SwiftUI + AppKit + SQLite

---

## 项目目标

macOS 剪贴板管理器，支持自动捕获、搜索、OCR、自动粘贴、Prompt Tools 集成

---

## 功能范围

### 已实现

- ✅ 剪贴板自动捕获（文本、图片）
- ✅ 全局快捷键呼出历史面板（Cmd+Option+V，可自定义）
- ✅ 底部悬浮窗 + 横向卡片浏览
- ✅ 实时搜索 + 图片 OCR 文本搜索
- ✅ 自动粘贴（CGEvent 模拟 Cmd+V）
- ✅ 敏感数据过滤（密码管理器等）
- ✅ 设置面板（NavigationSplitView 侧边栏，5 个 Tab）
- ✅ Liquid Glass 视觉效果（macOS 26 `.glassEffect()` + 降级）
- ✅ Prompt Tools 功能（划词工具集成）
- ✅ 删除功能（卡片悬停删除 + 菜单栏清空）
- ✅ 开机自启动（macOS 13+ SMAppService）

### 非目标

- ❌ 不支持文件捕获
- ❌ 不做云同步
- ❌ 不做多设备协同

## 核心模块

### 1. 数据库管理

**表结构** (`clipboard_history`):
- id: INTEGER PRIMARY KEY
- unique_id: TEXT UNIQUE (SHA256 hash)
- type: TEXT ('text' | 'image')
- text_content: TEXT
- image_data: BLOB (\u003c512KB)
- blob_path: TEXT (\u003e512KB)
- timestamp: INTEGER
- app_name: TEXT
- app_path: TEXT
- ocr_text: TEXT

**功能**:
- SHA256 去重
- 200 条上限管理（可调 50-500，FIFO）
- 大文件分离存储（\u003e512KB → blobs/ 目录）
- 后台 OCR 识别（Vision Framework）

### 2. 剪贴板监听

**轮询**: 0.75 秒检查 `NSPasteboard.changeCount`

**敏感数据过滤**:
- org.nspasteboard.TransientType
- org.nspasteboard.ConcealedType
- org.nspasteboard.AutoGeneratedType

**暂停机制**: 写入剪贴板时暂停 1 秒（避免自己捕获）

### 3. 悬浮窗管理

**窗口规范** (v0.2.1):
- 位置: 屏幕底部居中，距底部 20pt
- 高度: 280pt
- 圆角: 20pt（macOS 26 系统级应用标准）
- 背景: `.glassEffect(.regular)`
- 显示动画: `.snappy(duration: 0.4, extraBounce: 0.0)`
- 隐藏动画: `.smooth(duration: 0.3, extraBounce: 0.0)`

**卡片规范**:
- 尺寸: 180×180pt（正方形）
- 圆角: 20pt
- 间距: 12pt
- 色条: 3pt（文本蓝 #007AFF，图片紫 #AF52DE）
- 背景: `.thinMaterial`
- 悬停: 1.05x 放大 + 阴影加深

### 4. 全局快捷键

- 默认: Cmd+Option+V（可自定义）
- API: Carbon EventHotKey
- 冲突检测: 启动时临时注册测试
- 支持修饰键: Cmd、Option、Shift、Control 任意组合

### 5. 自动粘贴

- CGEvent 模拟 Cmd+V
- Accessibility 权限检查
- 焦点切换 + 0.3 秒延迟
- 参考 Maccy 实现

### 6. 搜索功能

- 实时过滤（300ms 防抖）
- 搜索范围: textContent || appName || ocrText
- 支持图片 OCR 文本搜索
- 文本操作: Cmd+A/C/V/X

### 7. 图片 OCR

- Vision Framework (VNRecognizeTextRequest)
- 后台异步执行（Task.detached）
- 语言支持: 中文简繁体、英文
- 精确模式 + 语言校正

### 8. 设置面板

**布局** (v0.5 - Liquid Glass 标准):
- NavigationSplitView（侧边栏 + 详情面板）
- 窗口尺寸: 最小 650×400pt，支持任意大小调整
- 侧边栏: 150-400pt 灵活宽度（理想 200pt）
- Form-based 内容布局，使用系统默认材质
- 遵循 Apple "Adopting Liquid Glass" 官方指南

**分类**:
1. 通用（历史上限、自动粘贴、开机自启）
2. 快捷键（快捷键录制器 + 冲突检测）
3. Prompt Tools（划词工具配置）
4. 隐私（密码管理器过滤、应用过滤列表）
5. 高级（重置所有设置）

## 架构设计

### Clean Architecture（整洁架构）

项目采用 Clean Architecture，实现业务逻辑与框架解耦：

**架构层次**（依赖方向：外层 → 内层）：

1. **Domain Layer（领域层）**
   - Protocols（接口定义）：AIServiceProtocol, ClipboardReader, NotificationServiceProtocol 等
   - Value Objects（值对象）：ToolID, KeyCombo, ClipboardContent
   - Errors（领域错误）：PromptToolError

2. **Use Case Layer（用例层）**
   - ExecutePromptTool：执行 Prompt 工具
   - RegisterToolHotKey：注册工具快捷键
   - AnalyzeAndRecommend：Smart AI 分析推荐

3. **Coordinator Layer（协调器层）**
   - PromptToolCoordinator：协调 Prompt 工具相关用例
   - SmartToolCoordinator：协调 Smart AI 相关用例

4. **Adapter Layer（适配器层）**
   - Services：NSPasteboardAdapter, OpenAIServiceAdapter, UserNotificationAdapter
   - Repositories：SQLitePromptToolRepository

5. **Infrastructure Layer（基础设施层）**
   - DI Container：DependencyContainer, DependencyEnvironment
   - Legacy Managers：DatabaseManager, HotKeyManager 等（逐步迁移中）

**核心原则**：
- **依赖倒置**：业务逻辑依赖接口，不依赖具体实现
- **单一职责**：每个类只有一个变化原因
- **接口隔离**：小而专注的接口定义
- **依赖注入**：通过构造器注入依赖，便于测试
