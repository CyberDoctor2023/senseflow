# SenseFlow：基于"输入流"的隐形 AI Agent

---

## 核心定位

SenseFlow 不是剪贴板工具。

它是一个基于"输入流"的隐形 AI Agent：
- 你的每次复制、每次划词 = 输入信号
- AI 理解这些信号，主动提供帮助
- 从被动存储 → 主动理解 → 智能执行

## 使用场景

### 场景 1：多段文字来回切换

写文档时需要在多个段落之间来回复制粘贴。

**传统方式**：来回切换窗口，复制被覆盖后只能重新找。

**SenseFlow**：
1. 连续复制多段内容
2. 按快捷键呼出历史（Cmd+Option+V）
3. 点击需要的内容，自动粘贴

[图片占位：多段文字切换演示]

---

### 场景 2：查找之前复制的内容

昨天复制了一个 API Key，今天需要用，但忘记存在哪里了。

**传统方式**：回忆、查找、可能找不到。

**SenseFlow**：
1. 按快捷键呼出历史
2. 搜索关键词（如 "api"）
3. 找到并点击，自动粘贴

[图片占位：搜索历史内容]

---

### 场景 3：图片 OCR 搜索

截图了很多界面，想找某个包含"设置"按钮的截图。

**传统方式**：一张张点开看。

**SenseFlow**：
1. 截图自动保存到历史
2. 搜索"设置"
3. 自动找到包含"设置"文字的截图

**技术细节**：
- OCR 引擎：Apple Vision Framework
- 支持语言：中文简繁体、英文
- 识别准确率：90%+
- 处理方式：后台异步，不阻塞界面

[图片占位：OCR 搜索演示]

---

### 场景 4：代码片段管理

开发时经常需要复制各种代码片段、命令、配置等。

**SenseFlow**：
1. 连续复制多段代码
2. 按快捷键查看历史
3. 点击需要的命令，自动粘贴到终端

**功能亮点**：
- 自动识别代码类型
- 保留格式和缩进
- 支持搜索代码内容
- 按应用分类（VS Code、Terminal 等）

[图片占位：代码片段管理]

### 场景 5：AI 工具集成（Prompt Tools）

需要用 AI 处理文本，比如翻译、润色、总结等。

**传统方式**：复制 → 打开 ChatGPT → 粘贴 → 输入提示词 → 等待 → 复制结果。

**SenseFlow**：
1. 选中文本，复制
2. 按快捷键呼出历史
3. 点击 AI 工具按钮，选择功能
4. 自动处理并粘贴结果

**支持的 AI 功能**：
- 🌐 翻译：中英互译
- ✍️ 润色：改进文字表达
- 📝 总结：提取关键信息
- 🔧 自定义：添加自己的 Prompt

## 常见问题

### Q1: 历史记录保存多久？

**A**: 默认保存最近 200 条记录，可在设置中调整（50-500 条）。采用 FIFO 策略，超过上限自动删除最旧的记录。

---

### Q2: 支持哪些系统版本？

**A**: macOS 14.0 (Sonoma) 及以上版本。Liquid Glass 效果需要 macOS 26.0+，低版本自动降级。

---

### Q3: 数据存储在哪里？

**A**:
- 数据库：`~/Library/Application Support/SenseFlow/clipboard.db`
- 大图片：`~/Library/Application Support/SenseFlow/blobs/`
- 所有数据本地存储，不上传云端

---

### Q4: 如何删除历史记录？

**A**:
- 单条删除：悬停卡片，点击删除按钮
- 清空所有：菜单栏 → 清空历史记录

---

### Q5: 自动粘贴需要什么权限？

**A**: 需要 Accessibility 权限。首次使用时会提示授权，在"系统设置 → 隐私与安全性 → 辅助功能"中勾选 SenseFlow。

---

### Q6: 支持哪些 AI 服务？

**A**: 支持 OpenAI 兼容的所有服务：
- OpenAI (GPT-4)
- Claude (Anthropic)
- DeepSeek
- 其他兼容服务

## 快速开始

### 第一次使用

1. **启动应用**：双击 SenseFlow
2. **授权权限**：允许 Accessibility 权限（可选）
3. **复制内容**：复制任意文本或图片
4. **呼出窗口**：按 Cmd+Option+V
5. **选择内容**：点击卡片自动粘贴

[图片占位：快速开始流程图]

---

### 推荐设置

1. **开机自启动**：设置 → 通用 → 勾选
2. **自定义快捷键**：设置 → 快捷键 → 录制
3. **调整历史上限**：设置 → 通用 → 历史记录上限
4. **配置 AI 工具**：设置 → Prompt Tools → 添加 API Key
