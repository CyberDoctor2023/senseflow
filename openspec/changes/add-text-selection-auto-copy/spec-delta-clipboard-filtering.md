# Spec Delta: clipboard-filtering

## Metadata
- **Capability ID**: clipboard-filtering
- **Type**: new
- **Change ID**: add-text-selection-auto-copy
- **Version**: 0.5.0

## Summary
统一的剪贴板过滤管理器，集中管理所有过滤规则（敏感数据、密码管理器、应用黑名单），为 ClipboardMonitor 和 TextSelectionMonitor 提供统一的过滤接口。

## Requirements

### REQ-CF-001: 单例模式
- **Priority**: MUST
- **Description**: ClipboardFilterManager 必须使用单例模式（shared instance）
- **Rationale**: 全局只需一个过滤管理器实例，避免重复配置和状态不一致

### REQ-CF-002: 公开过滤接口
- **Priority**: MUST
- **Description**: 提供公开方法 `shouldFilter(pasteboardTypes:appBundleID:appName:) -> FilterResult`
- **Rationale**: 两个监听器需要通过统一接口调用过滤逻辑
- **Parameters**:
  - `pasteboardTypes`: 可选，剪贴板数据类型数组
  - `appBundleID`: 可选，应用 Bundle ID
  - `appName`: 可选，应用名称
- **Return**: FilterResult 结构体（shouldFilter: Bool, reason: FilterReason?）

### REQ-CF-003: 敏感数据过滤
- **Priority**: MUST
- **Description**: 检测并过滤敏感数据类型（TransientType、ConcealedType 等）
- **Rationale**: 保护用户隐私，避免保存密码等敏感信息
- **Implementation**: 从 ClipboardMonitor 迁移现有的 `sensitiveTypes` 定义

### REQ-CF-004: 密码管理器过滤
- **Priority**: MUST
- **Description**: 检测并过滤密码管理器应用的剪贴板内容
- **Rationale**: 密码管理器的内容通常是敏感的，不应保存
- **Implementation**:
  - 内置密码管理器 Bundle ID 列表（1Password、LastPass、Bitwarden 等）
  - 读取 UserDefaults `filter_password_managers` 开关
  - 只有开关开启时才过滤

### REQ-CF-005: 应用黑名单过滤
- **Priority**: MUST
- **Description**: 检测并过滤用户自定义黑名单中的应用
- **Rationale**: 实现现有设置界面中的应用过滤功能（之前未实现）
- **Implementation**:
  - 读取 UserDefaults `filter_app_list`（换行符分隔的字符串）
  - 支持 Bundle ID 匹配（精确匹配）
  - 支持应用名称匹配（不区分大小写）

### REQ-CF-006: 过滤优先级
- **Priority**: MUST
- **Description**: 按优先级顺序检查过滤规则
- **Order**:
  1. 敏感数据类型（最高优先级）
  2. 密码管理器（中优先级）
  3. 用户黑名单（最低优先级）
- **Rationale**: 敏感数据过滤最重要，应优先检查

### REQ-CF-007: 过滤原因返回
- **Priority**: SHOULD
- **Description**: 返回 FilterResult 包含过滤原因（FilterReason 枚举）
- **Rationale**: 便于调试和日志记录
- **Values**:
  - `.sensitiveData` - 敏感数据类型
  - `.passwordManager` - 密码管理器
  - `.blacklistedApp` - 黑名单应用

### REQ-CF-008: 可选参数支持
- **Priority**: MUST
- **Description**: 所有参数都是可选的，支持部分信息缺失的场景
- **Rationale**: 某些场景可能无法获取完整的应用信息
- **Behavior**:
  - 如果 `pasteboardTypes` 为 nil，跳过敏感数据检查
  - 如果 `appBundleID` 和 `appName` 都为 nil，跳过应用级别检查

## Scenarios

### SCENARIO-CF-001: 过滤密码管理器内容
**Given**: 用户在 1Password 中复制密码
**When**: ClipboardMonitor 调用 `shouldFilter(pasteboardTypes: types, appBundleID: "com.agilebits.onepassword7", appName: "1Password")`
**Then**: 返回 `FilterResult(shouldFilter: true, reason: .passwordManager)`

### SCENARIO-CF-002: 过滤黑名单应用
**Given**: 用户在设置中添加 "com.apple.Terminal" 到黑名单
**When**: ClipboardMonitor 调用 `shouldFilter(pasteboardTypes: types, appBundleID: "com.apple.Terminal", appName: "Terminal")`
**Then**: 返回 `FilterResult(shouldFilter: true, reason: .blacklistedApp)`

### SCENARIO-CF-003: 过滤敏感数据类型
**Given**: 剪贴板包含 TransientType 数据
**When**: ClipboardMonitor 调用 `shouldFilter(pasteboardTypes: [.init("org.nspasteboard.TransientType")], appBundleID: nil, appName: nil)`
**Then**: 返回 `FilterResult(shouldFilter: true, reason: .sensitiveData)`

### SCENARIO-CF-004: 通过所有过滤规则
**Given**: 用户在 TextEdit 中复制普通文本
**When**: ClipboardMonitor 调用 `shouldFilter(pasteboardTypes: [.string], appBundleID: "com.apple.TextEdit", appName: "TextEdit")`
**Then**: 返回 `FilterResult(shouldFilter: false, reason: nil)`

### SCENARIO-CF-005: 密码管理器过滤开关关闭
**Given**: UserDefaults `filter_password_managers` = false
**When**: ClipboardMonitor 调用 `shouldFilter(pasteboardTypes: types, appBundleID: "com.agilebits.onepassword7", appName: "1Password")`
**Then**: 返回 `FilterResult(shouldFilter: false, reason: nil)`

### SCENARIO-CF-006: 应用名称匹配（不区分大小写）
**Given**: 用户在设置中添加 "terminal" 到黑名单
**When**: ClipboardMonitor 调用 `shouldFilter(pasteboardTypes: types, appBundleID: nil, appName: "Terminal")`
**Then**: 返回 `FilterResult(shouldFilter: true, reason: .blacklistedApp)`

### SCENARIO-CF-007: 部分信息缺失
**Given**: 无法获取应用信息
**When**: TextSelectionMonitor 调用 `shouldFilter(pasteboardTypes: nil, appBundleID: nil, appName: nil)`
**Then**: 返回 `FilterResult(shouldFilter: false, reason: nil)`

## Dependencies

### Internal Dependencies
- UserDefaults（读取配置）
- NSPasteboard.PasteboardType（类型定义）

### External Dependencies
- 无

## Notes

### Implementation Notes
- 内置密码管理器列表应包含常见的密码管理器：
  - 1Password: `com.agilebits.onepassword7`, `com.agilebits.onepassword8`
  - LastPass: `com.lastpass.LastPass`
  - Bitwarden: `com.bitwarden.desktop`
  - Dashlane: `com.dashlane.dashlanephonefinal`
  - Keeper: `com.callpod.keeper`

- 敏感数据类型列表（从 ClipboardMonitor 迁移）：
  - `org.nspasteboard.TransientType`
  - `org.nspasteboard.ConcealedType`
  - `org.nspasteboard.AutoGeneratedType`

### Testing Notes
- 单元测试应覆盖所有过滤规则
- 测试优先级顺序（敏感数据 > 密码管理器 > 黑名单）
- 测试可选参数的各种组合

### Migration Notes
- ClipboardMonitor 的 `sensitiveTypes` 和 `containsSensitiveData()` 将被移除
- 现有的应用过滤功能将首次实现（之前只有 UI，没有实现）
